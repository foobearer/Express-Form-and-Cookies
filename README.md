# Express-Form-and-Cookies
Getting data from form and cookies

I've written a blog to demonstrate a step-by-step process on how you'll be accomplishing this.
Check this [blog link](https://medium.com/javascript-in-plain-english/accessing-data-from-cookies-and-forms-using-express-node-js-c140c9f59f6d)


## Install Node Packages
To install all of the dependencies needed, you have to run the command: 

`npm install`

## Running the Application
To run the application you have to run the command:

`npm start`

## App's Preview
You will be creating a login form and will be redirected to a welcome page once it passes the requirements.
